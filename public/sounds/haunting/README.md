# Haunting Audio Files

This directory contains audio files for the advanced haunting system's four-layer audio architecture.

## Directory Structure

```
haunting/
├── ambient/          # Ambient layer sounds
│   ├── distant-scream.mp3
│   ├── distant-scream-1.mp3
│   └── distant-scream-2.mp3
├── effects/          # Effects layer sounds
│   ├── phantom-typing.mp3
│   └── hdd-grinding.mp3
├── whispers/         # Whispers layer sounds (future)
│   └── generic-whisper.mp3
└── tension/          # Tension layer sounds (future)
    └── heartbeat.mp3
```

## Audio Requirements

### Ambient Layer
- **distant-scream.mp3**: Distant, echoing scream sound (3-5 seconds)
- **distant-scream-1.mp3**: Variant 1 of distant scream
- **distant-scream-2.mp3**: Variant 2 of distant scream

### Effects Layer
- **phantom-typing.mp3**: Keyboard typing sounds (2-4 seconds)
- **hdd-grinding.mp3**: Hard drive grinding/seeking sounds (3-8 seconds)

### Whispers Layer (Task 7)
- **generic-whisper.mp3**: Unintelligible whispers

### Tension Layer (Task 7)
- **heartbeat.mp3**: Single heartbeat sound for looping

## Audio Specifications

- **Format**: MP3
- **Bitrate**: 64-128 kbps (optimized for web)
- **Sample Rate**: 44.1 kHz
- **Channels**: Mono or Stereo
- **Normalization**: -3dB peak to prevent clipping
