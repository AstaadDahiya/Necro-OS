# Spirit Board Assets

This directory contains graphics for the Spirit Board (Ouija board) application.

## Files

- `ouija-board.svg` - Complete Ouija board layout with letters A-Z, numbers 0-9, YES, NO, and GOODBYE
- `planchette.svg` - Semi-transparent planchette pointer with crosshairs for letter selection

## Design Details

### Ouija Board
- Dark wood-textured background with mystical gradient
- Golden letters with glow effect for supernatural atmosphere
- Decorative border and corner symbols
- Layout optimized for 800x600 window size

### Planchette
- Teardrop/heart shape traditional to Ouija boards
- Semi-transparent wood texture
- Center viewing hole with crosshairs for precise letter targeting
- Glow effect for supernatural appearance

## Usage

These graphics are used by the SpiritBoard.vue component for AI-powered supernatural communication. The planchette animates across the board to spell out responses from the Gemini API.
